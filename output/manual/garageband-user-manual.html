1. 화면 구조와 기본 개념

Transport Bar재생, 정지, 녹음, 사이클, Undo/Redo, BPM, 박자표, Key, 메트로놈, 튜너, 저장, Export가 모여 있는 앱의 조종석입니다.
Arrangement Timeline트랙별로 클립을 놓고 곡의 구조를 만드는 영역입니다. Tracks 보기와 Live Loops 보기를 전환할 수 있습니다.
Studio PanelLibrary, Loops, Audio, Smart, Lesson 탭이 있으며 소리 선택, 녹음, 믹서, 교육 기능을 제공합니다.
Clip Editor선택한 클립의 세부 편집 영역입니다. MIDI면 Piano Roll/Touch/Drummer, 오디오면 파형 편집이 나타납니다.

프로젝트, 트랙, 클립의 관계

  1. 프로젝트는 한 곡 전체입니다. BPM, Key, 박자표, 트랙, 클립, 수업 정보, 라이브 루프 정보를 포함합니다.
  2. 트랙은 악기나 오디오가 놓이는 줄입니다. 비트, 베이스, 건반, 오디오 트랙처럼 역할을 나눠 씁니다.
  3. 클립은 트랙 위에 놓이는 색상 블록입니다. 루프, MIDI, 오디오, Drummer 클립이 있습니다.
  4. 클립을 선택하면 하단 Clip Editor가 해당 클립에 맞는 편집 화면으로 바뀝니다.

2. Transport와 프로젝트 설정

  1. 새 프로젝트: 오른쪽 상단 New를 누르면 새 곡을 시작합니다.
  2. 프로젝트 이름: 상단의 프로젝트 이름 입력칸을 클릭해 곡 제목을 바꿉니다.
  3. Tempo: 숫자 입력 또는 Tap으로 BPM을 정합니다. 연주하면서 Tap을 누르면 감각적인 템포를 잡을 수 있습니다.
  4. 박자표와 Key: 드롭다운에서 4/4, 3/4, 6/8 등과 C, Am 같은 조성을 고릅니다.
  5. Metronome / Count-in: 녹음 전 박자 안내가 필요하면 Metro와 Count-in을 켭니다.
  6. 튜너: 마이크 권한을 허용하면 Tuner 모드에서 입력 음정을 확인할 수 있습니다.
  7. Save: 로컬 저장소에 현재 프로젝트를 저장합니다. 앱은 자동 저장도 수행하지만 중요한 작업 전후에는 Save를 누르는 습관이 좋습니다.
녹음 전 추천 설정BPM, Key, Count-in, Metronome, 녹음할 Audio 트랙의 Record enable을 먼저 확인하세요.

3. Timeline에서 클립 배치하기

트랙 추가와 기본 편곡

  1. Timeline 상단의 + 드럼, Drummer, + 악기 버튼으로 트랙이나 클립을 추가합니다.
  2. 오른쪽 Loops 탭에서 루프를 드래그해 원하는 트랙과 마디 위치에 놓습니다.
  3. 클립을 클릭하면 선택되고, 드래그하면 위치가 바뀝니다.
  4. 클립 왼쪽/오른쪽 핸들을 드래그하면 시작점과 길이가 바뀝니다.
  5. 루프 반복 핸들을 사용하면 같은 패턴을 더 길게 반복할 수 있습니다.
  6. 스냅 단위를 1/4, 1/2, 1박, 1마디로 바꾸면 이동과 길이 조절이 그리드에 맞춰집니다.
버튼/영역하는 일추천 상황
Tracks / Live Loops편곡형 타임라인과 라이브 루프 그리드를 전환곡 구조를 만들 때 Tracks, 즉흥 트리거는 Live Loops
겹침 방지클립 이동 시 같은 트랙에서 겹치지 않게 보정처음 편곡하거나 수업용 과제를 만들 때
Cycle반복 재생 범위 켜기/끄기특정 구간 연습, 녹음, 구간 Export
확대 슬라이더Timeline 가로 배율 조절세밀한 MIDI/오디오 위치 편집

4. Loop Browser와 Library

Loop Browser 사용 절차

  1. 오른쪽 Studio Panel에서 Loops 탭을 엽니다.
  2. 검색창에 drum, bass, synth 같은 단어를 입력하거나 카테고리 버튼을 누릅니다.
  3. 장르와 무드 드롭다운으로 결과를 좁힙니다.
  4. 재생 버튼으로 미리 들어본 뒤 + 버튼을 누르거나 클립을 Timeline으로 드래그합니다.
  5. Live Loops 보기에서는 빈 셀로 드래그해 즉흥 연주용 셀을 만들 수 있습니다.

Library에서 악기 고르기

  1. 오른쪽 Library 탭을 엽니다.
  2. 선택된 트랙의 역할에 맞는 악기 패치를 고릅니다. 드럼 트랙은 Drum Kit, 멜로디/화성 트랙은 Keys나 Synth 계열이 어울립니다.
  3. 악기를 바꾸면 이후 MIDI 클립 재생 톤이 바뀝니다.
  4. 베이스, 멜로디, 화성 트랙을 분리하면 Mixer에서 밸런스를 잡기 쉽습니다.

5. MIDI, Touch, Drummer 편집

MIDI 클립 만들기와 편집

  1. Timeline에서 악기 트랙을 선택하고 하단 Clip Editor의 미디 클립 버튼을 누릅니다.
  2. Piano Roll에서 원하는 위치를 클릭하거나 드래그해 노트를 입력합니다.
  3. 노트를 드래그해 음높이와 시작 박자를 조절합니다.
  4. 노트 끝을 잡아 길이를 늘리거나 줄입니다.
  5. Quantize를 사용해 애매한 타이밍을 스냅 단위에 맞춥니다.
  6. Scale Lock을 켜면 프로젝트 Key/Scale에 어울리는 음 위주로 작업할 수 있습니다.

Touch Instruments

  1. MIDI 클립을 선택한 뒤 하단 편집기에서 Touch를 선택합니다.
  2. Keyboard, Beat Sequencer, Smart Drums, Chord Strips 중 필요한 입력 방식을 고릅니다.
  3. Preview 상태에서는 소리만 확인하고, REC 상태 또는 쓰기 동작에서는 노트가 클립에 기록됩니다.
  4. 코드 스트립은 화성 진행을 빠르게 만들 때, Beat Sequencer는 드럼 패턴을 세밀하게 만들 때 유용합니다.

Drummer 클립

  1. Timeline 상단 Drummer 버튼을 눌러 Drummer 클립을 만듭니다.
  2. 하단 편집기에서 Drummer 모드가 열리면 프리셋, Complexity, Loudness, Swing, Fills를 조정합니다.
  3. 설정을 바꾸면 MIDI 드럼 패턴이 재생성됩니다.
  4. 수업용 프로젝트에서는 Drummer로 기본 리듬을 만든 뒤 베이스와 멜로디를 얹는 방식이 빠릅니다.
MIDI 편집 팁드럼은 짧고 반복적인 4마디, 베이스는 루트음을 중심으로 8마디, 멜로디는 2-4마디 모티프를 먼저 만들면 곡 구조를 잡기 쉽습니다.

6. 오디오 녹음과 파형 편집

오디오 녹음 절차

  1. 오른쪽 Studio Panel에서 Audio 탭을 엽니다.
  2. Track 버튼으로 오디오 트랙을 추가합니다.
  3. 녹음할 트랙에서 Record enable을 켭니다. Timeline 트랙 메뉴 또는 Mixer에서도 켤 수 있습니다.
  4. 상단 Count-in과 Metronome을 설정합니다.
  5. 상단 Record 버튼을 누르면 녹음이 시작됩니다. Stop을 누르면 녹음이 종료되고 오디오 클립이 만들어집니다.
  6. 하단 Clip Editor에서 파형을 보며 Trim, Fade, Gain을 조정합니다.

Take와 Comp

  1. 같은 구간을 여러 번 녹음하면 Take로 관리할 수 있습니다.
  2. 각 Take의 좋은 구간을 선택해 Take Section을 구성합니다.
  3. Create Comp를 사용하면 선택된 구간을 하나의 편집 가능한 오디오 클립으로 만들 수 있습니다.
  4. Comp 후에도 Split, Normalize, Fade 편집을 이어서 적용할 수 있습니다.

7. Mixer, Smart Controls, Automation

Mixer 기본 조정

  1. 오른쪽 Studio Panel에서 Smart 탭을 엽니다.
  2. 각 Channel Strip에서 Volume fader로 소리 크기를 맞춥니다.
  3. Pan으로 좌우 위치를 정합니다. 베이스와 킥은 중앙에 두는 편이 안정적입니다.
  4. Mute는 잠시 숨기기, Solo는 특정 트랙만 듣기에 사용합니다.
  5. Reverb/Delay Send로 공간감을 더합니다.
  6. Master Limiter를 켜면 Export 시 갑작스러운 피크를 줄이는 데 도움이 됩니다.

Automation 작성 절차

  1. Timeline 트랙 헤더의 A 버튼을 눌러 Automation Lane을 엽니다.
  2. Lane 왼쪽 위 드롭다운에서 Volume, Pan, Send Reverb, Send Delay 중 하나를 선택합니다.
  3. Lane 위를 클릭해 포인트를 추가합니다.
  4. 포인트를 드래그해 박자와 값을 조절합니다.
  5. 포인트를 삭제하려면 포인트를 선택하거나 보조 클릭 메뉴를 사용합니다.
  6. Export에는 Automation 움직임이 반영됩니다.

8. Live Loops 즉흥 연주

  1. Timeline 상단의 Live Loops 버튼을 눌러 그리드 보기로 전환합니다.
  2. 오른쪽 Loops 탭에서 루프를 빈 셀로 드래그합니다.
  3. 셀을 클릭하면 해당 루프가 다음 마디 또는 지정된 Quantize 지점에 큐잉됩니다.
  4. Scene 헤더를 클릭하면 같은 열의 셀이 한 번에 큐잉됩니다.
  5. Stop live loops 버튼으로 실행 중인 셀을 모두 멈춥니다.
  6. 즉흥으로 좋은 조합을 찾은 뒤 Tracks 보기에서 편곡으로 정리하면 빠르게 곡을 완성할 수 있습니다.

9. Lesson, Review, Teacher 흐름

Student학생 프로필, 과제 시작, 제출 흐름을 다룹니다. 수업용 프로젝트는 Lesson 미션과 연결됩니다.
Review곡 길이, 트랙 구성, 밸런스, 루브릭 상태를 점검합니다. 제출 전 체크리스트로 쓰기 좋습니다.
Teacher반, 학생, 과제, 피드백을 관리합니다. 로컬 저장소, Mock Cloud, Supabase 모드를 선택할 수 있습니다.

10. Share Export와 파일 관리

Mix 내보내기

  1. 상단 오른쪽 Export를 누릅니다.
  2. Format에서 WAV 또는 MP3를 고릅니다. MP3는 현재 WAV 폴백 메시지가 표시될 수 있습니다.
  3. Quality에서 Standard 또는 High를 고릅니다.
  4. Range에서 Full 또는 Cycle을 고릅니다. Cycle은 사이클 영역이 켜져 있어야 사용할 수 있습니다.
  5. Mix를 누르면 전체 믹스 파일을 다운로드합니다.

Stems ZIP 내보내기

  1. Share 모달에서 Range를 확인합니다.
  2. Stems ZIP을 누릅니다.
  3. 각 트랙이 개별 WAV 파일로 렌더링되어 ZIP 안에 들어갑니다.
  4. 외부 DAW에서 믹싱하거나 선생님에게 트랙별 피드백을 받을 때 사용합니다.

프로젝트 파일 저장/불러오기

  1. Project를 누르면 현재 프로젝트가 .webband.json 파일로 다운로드됩니다.
  2. 다른 브라우저나 다른 컴퓨터에서 이어서 작업하려면 같은 Share 모달의 Import를 누릅니다.
  3. 가져오기 전에 현재 프로젝트가 저장되어 자동 백업 역할을 합니다.
  4. 불러온 프로젝트는 현재 앱 버전에 맞춰 자동 보정됩니다.
파일내용사용 목적
WAV전체 믹스 오디오제출, 공유, 영상 편집, 마스터 확인
MP3 선택현재는 WAV 폴백MP3 UI 흐름 유지, 인코더 미지원 환경 보호
Stems ZIP트랙별 WAV 묶음외부 믹싱, 협업, 교사용 피드백
.webband.json프로젝트 구조 데이터백업, 이동, 복원, 과제 전달

11. 10분 첫 곡 만들기

  1. New로 새 프로젝트를 만들고 이름을 정합니다.
  2. BPM 100-130 사이에서 하나를 정하고 Key를 C 또는 Am으로 시작합니다.
  3. Loops 탭에서 드럼 루프를 찾아 첫 번째 트랙에 놓습니다.
  4. 베이스 루프를 두 번째 트랙에 놓고 드럼과 길이를 맞춥니다.
  5. 악기 트랙을 만들고 MIDI 클립 또는 코드 스트립으로 간단한 코드 진행을 추가합니다.
  6. 필요하면 Audio 탭에서 보컬이나 악기를 녹음합니다.
  7. Smart 탭에서 트랙별 볼륨을 조정하고 Master Limiter를 확인합니다.
  8. Review에서 제출 가능성을 확인합니다.
  9. Export -> Mix로 WAV를 받고, 백업용 Project 파일도 저장합니다.

12. 단축키와 문제 해결

단축키동작사용 팁
Space재생/일시정지입력창에 커서가 없을 때 동작합니다.
R녹음 시작/종료녹음할 오디오 트랙의 Record enable을 먼저 켜세요.
Enter재생 위치를 처음으로 이동반복 확인 전 빠르게 처음으로 돌아갈 때 씁니다.
Ctrl + ZUndo트랙/클립/노트/믹서 편집을 되돌립니다.
Ctrl + YRedoUndo한 작업을 다시 적용합니다.
소리가 안 날 때Master volume, 트랙 Mute/Solo, 브라우저 탭 음소거, 시스템 출력 장치, 클립 위치를 차례로 확인하세요.
녹음이 안 될 때마이크 권한을 허용했는지, Audio 트랙이 있는지, 해당 트랙의 Record enable이 켜져 있는지 확인하세요.
Export가 예상과 다를 때Cycle 범위가 켜져 있으면 Range가 Cycle일 수 있습니다. Full로 바꾸거나 사이클을 끄고 다시 내보내세요.
